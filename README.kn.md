## 🚀 ಹೊಸ ವೈಶಿಷ್ಟ್ಯಗಳು ಮತ್ತು ಹಾರ್ಡ್‌ವೇರ್ ಬೆಂಬಲ (v2.6.1)

ಈ ಆವೃತ್ತಿಯು ಆಧುನಿಕ ಹಾರ್ಡ್‌ವೇರ್‌ಗಾಗಿ ಗಮನಾರ್ಹ ಆಪ್ಟಿಮೈಸೇಶನ್ ಮತ್ತು ಬೆಂಬಲವನ್ನು ತರುತ್ತದೆ:

### 💻 MSI Modern 14 C7M ಮತ್ತು AMD 7000 ಸರಣಿ
- **ಆಪ್ಟಿಮೈಸ್ಡ್ AMD 7530U ಕಾರ್ಯಕ್ಷಮತೆ**: ಶಕ್ತಿ ಮತ್ತು ಬ್ಯಾಟರಿ ಬಾಳಿಕೆಯ ನಡುವಿನ ಪರಿಪೂರ್ಣ ಸಮತೋಲನಕ್ಕಾಗಿ `amd-pstate-epp` ಮತ್ತು `auto-cpufreq` ಅನ್ನು ಪೂರ್ಣವಾಗಿ ಬಳಸಿಕೊಳ್ಳುತ್ತದೆ.
- **ಬ್ಯಾಟರಿ ಆರೋಗ್ಯ ನಿರ್ವಹಣೆ**: `msi-ec` ಮೂಲಕ **MSI ಬ್ಯಾಟರಿ ಥ್ರೆಶೋಲ್ಡ್‌ಗಳಿಗೆ** ಸ್ಥಳೀಯ ಬೆಂಬಲ, ಪ್ಲಗ್ ಇನ್ ಮಾಡಿದಾಗ ಚಾರ್ಜಿಂಗ್ ಅನ್ನು 80% ಗೆ ಸೀಮಿತಗೊಳಿಸುವ ಮೂಲಕ ನಿಮ್ಮ ಬ್ಯಾಟರಿಯನ್ನು ರಕ್ಷಿಸುತ್ತದೆ.
- **ಸುಧಾರಿತ ಪವರ್ ಸ್ಕೇಲಿಂಗ್**: ಪವರ್ ಸ್ಥಿತಿಯ ಆಧಾರದ ಮೇಲೆ `performance` ಮತ್ತು `powersave` ಮೋಡ್‌ಗಳ ನಡುವೆ ಸ್ವಯಂಚಾಲಿತ ಸ್ವಿಚಿಂಗ್.
- **ನೆಕ್ಸ್ಟ್-ಜೆನ್ ಗ್ರಾಫಿಕ್ಸ್**: VA-API ಹಾರ್ಡ್‌ವೇರ್ ವೇಗವರ್ಧನೆ, ROCm ಮತ್ತು Vulkan ಪರಿಕರಗಳೊಂದಿಗೆ ಪೂರ್ಣ `amdgpu` ಬೆಂಬಲವನ್ನು ಮೊದಲೇ ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗಿದೆ.

### 🎨 ವೈವಿಧ್ಯಮಯ ಶೆಲ್ ಅನುಭವಗಳು
`variables.nix` ನಲ್ಲಿ `barChoice` ಅನ್ನು ಹೊಂದಿಸುವ ಮೂಲಕ ನಿಮ್ಮ ಆದ್ಯತೆಯ ಡೆಸ್ಕ್‌ಟಾಪ್ ಅನುಭವವನ್ನು ಆರಿಸಿ:
- **Noctalia**: ಸಂಯೋಜಿತ ಸಿಸ್ಟಮ್ ನಿಯಂತ್ರಣಗಳೊಂದಿಗೆ ಆಧುನಿಕ, ವೈಶಿಷ್ಟ್ಯ-ಸಮೃದ್ಧ ಶೆಲ್.
- **Caelestia**: ಒಂದು ಸೊಗಸಾದ ಮತ್ತು ಹಗುರವಾದ ಪರ್ಯಾಯ.
- **DMS (DankMaterialShell)**: ನಯವಾದ, ಆಧುನಿಕ ನೋಟಕ್ಕಾಗಿ ಮೆಟೀರಿಯಲ್-ಡಿಸೈನ್ ಪ್ರೇರಿತ ಶೆಲ್.
- **Waybar**: ಕ್ಲಾಸಿಕ್, ಹೆಚ್ಚು ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದಾದ ಸ್ಟೇಟಸ್ ಬಾರ್.

### 🔧 ಸಿಸ್ಟಮ್ ಸುಧಾರಣೆಗಳು
- **ಇತ್ತೀಚಿನ ಲಿನಕ್ಸ್ ಕರ್ನಲ್**: ಅತ್ಯುತ್ತಮ ಹಾರ್ಡ್‌ವೇರ್ ಹೊಂದಾಣಿಕೆಗಾಗಿ ಈಗ **7.x ಕರ್ನಲ್** ಸಾಲಿನಲ್ಲಿ ಚಾಲನೆಯಲ್ಲಿದೆ.

## 🏗️ ಅನುಸ್ಥಾಪನೆ: ಸ್ವಯಂಚಾಲಿತ ಮತ್ತು ಹಸ್ತಚಾಲಿತ

MayankOS ಪ್ರಾರಂಭಿಸಲು ಎರಡು ಪ್ರಾಥಮಿಕ ವಿಧಾನಗಳನ್ನು ನೀಡುತ್ತದೆ:
1. **ಸ್ವಯಂಚಾಲಿತ ಸ್ಕ್ರಿಪ್ಟ್ (ಹೊಸ ಬಳಕೆದಾರರಿಗೆ ಶಿಫಾರಸು ಮಾಡಲಾಗಿದೆ)**: ನಿಮಗಾಗಿ ಎಲ್ಲವನ್ನೂ ನಿರ್ವಹಿಸುವ ವೇಗದ, ಒಂದು-ಕಮಾಂಡ್ ಇನ್‌ಸ್ಟಾಲ್. ವರ್ಕಿಂಗ್ ಡೆಸ್ಕ್‌ಟಾಪ್ ಪಡೆಯಲು ಇದು ಅತ್ಯಂತ ವೇಗವಾದ ಮಾರ್ಗವಾಗಿದೆ.
2. **ಹಸ್ತಚಾಲಿತ ಅನುಸ್ಥಾಪನೆ**: ತಮ್ಮ ಸಿಸ್ಟಂನ ಪ್ರತಿಯೊಂದು ಅಂಶದ ಮೇಲೆ ಸಂಪೂರ್ಣ ನಿಯಂತ್ರಣವನ್ನು ಬಯಸುವ ಬಳಕೆದಾರರಿಗಾಗಿ. ನಿಮ್ಮ ಮೊದಲ ಮರುನಿರ್ಮಾಣಕ್ಕಿಂತ *ಮೊದಲು* ನಿಮ್ಮ ಪರಿಕರಗಳನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಲು ನೀವು ಬಯಸಿದರೆ ಹಸ್ತಚಾಲಿತ ಅನುಸ್ಥಾಪನೆಯು ಉತ್ತಮವಾಗಿದೆ.

## ⚡ ವೃತ್ತಿಪರ VLSI ಮತ್ತು ಹಾರ್ಡ್‌ವೇರ್ ಎಂಜಿನಿಯರಿಂಗ್ ವರ್ಕ್‌ಸ್ಟೇಷನ್

MayankOS ಅನ್ನು ನಿರ್ದಿಷ್ಟವಾಗಿ **VLSI ಮತ್ತು ಹಾರ್ಡ್‌ವೇರ್ ಎಂಜಿನಿಯರಿಂಗ್** ಗಾಗಿ ಉನ್ನತ-ಕಾರ್ಯಕ್ಷಮತೆಯ, ವೃತ್ತಿಪರ ವರ್ಕ್‌ಸ್ಟೇಷನ್ ಆಗಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

- **ಇದು ಏಕೆ ಪರಿಪೂರ್ಣವಾಗಿದೆ**: ಇದು ಸಮಗ್ರ ಪರಿಕರಗಳ ಸೂಟ್‌ನೊಂದಿಗೆ ಪೂರ್ವ-ಸಂರಚಿತವಾಗಿ ಬರುತ್ತದೆ:
  - **HDL ಸಿಮ್ಯುಲೇಶನ್**: `iverilog`, `verilator`, `gtkwave`.
  - **ಸಿಂಥೆಸಿಸ್ ಮತ್ತು ಭೌತಿಕ ವಿನ್ಯಾಸ**: `yosys`, `magic-vlsi`, `klayout`, `openroad`.
  - **FPGA ಮತ್ತು ಎಂಬೆಡೆಡ್ ಅಭಿವೃದ್ಧಿ**: `nextpnr`, `icestorm`, `dfu-util`, `qemu`.
  - **PCB ಮತ್ತು ಸ್ಕೀಮ್ಯಾಟಿಕ್ ವಿನ್ಯಾಸ**: `kicad`, `ngspice`.
- **ಕಸ್ಟಮೈಸ್ ಮಾಡುವುದು ಹೇಗೆ**: ನಿಮಗೆ ಈ ಪರಿಕರಗಳು ಅಗತ್ಯವಿಲ್ಲದಿದ್ದರೆ, ನಿಮ್ಮ `nixos-rebuild` ಅನ್ನು ಚಲಾಯಿಸುವ ಮೊದಲು `modules/core/packages.nix` ನಲ್ಲಿನ `# --- VLSI & Hardware Engineering ---` ಬ್ಲಾಕ್ ಅನ್ನು ಕಾಮೆಂಟ್ ಮಾಡಬಹುದು ಅಥವಾ ತೆಗೆದುಹಾಕಬಹುದು.
- **ಭವಿಷ್ಯಕ್ಕೆ ಸಿದ್ಧ**: ಇದು ಕೇವಲ ಪ್ರಾರಂಭವಾಗಿದೆ; MayankOS ಅನ್ನು ಹಾರ್ಡ್‌ವೇರ್ ವಿನ್ಯಾಸಕರಿಗೆ ಅಂತಿಮ ವೇದಿಕೆಯನ್ನಾಗಿ ಮಾಡಲು ಭವಿಷ್ಯದ ನವೀಕರಣಗಳಿಗಾಗಿ ಹೆಚ್ಚಿನ ವಿಶೇಷ VLSI ಮತ್ತು EDA ಪರಿಕರಗಳನ್ನು ಯೋಜಿಸಲಾಗಿದೆ।

## 🌐 ವೆಬ್ ಬ್ರೌಸರ್ ಆಯ್ಕೆ ಮತ್ತು ಗ್ರಾಹಕೀಕರಣ

### ಮೈಕ್ರೋಸಾಫ್ಟ್ ಎಡ್ಜ್ ಏಕೆ?

ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ, MayankOS ಈಗ **Microsoft Edge** ಅನ್ನು ಬಳಸುತ್ತದೆ. ಲಿನಕ್ಸ್ ಸಮುದಾಯವು ಫೈರ್‌ಫಾಕ್ಸ್, ಜೆನ್ ಅಥವಾ ಬ್ರೇವ್‌ನಂತಹ ಬ್ರೌಸರ್‌ಗಳಿಗೆ ಬಲವಾದ ಆದ್ಯತೆಗಳನ್ನು ಹೊಂದಿದೆ ಎಂದು ನಾವು ಗುರುತಿಸುತ್ತೇವೆ. ಆದಾಗ್ಯೂ, ಈ ವರ್ಕ್‌ಸ್ಟೇಷನ್‌ಗಾಗಿ ಎಡ್ಜ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಲಾಗಿದೆ ಏಕೆಂದರೆ:
- **ಹೊಂದಾಣಿಕೆ**: ಇದು ವೃತ್ತಿಪರ ಹಾರ್ಡ್‌ವೇರ್ ದಸ್ತಾವೇಜನ್ನು ಪೋರ್ಟಲ್‌ಗಳು ಮತ್ತು ವೆಬ್-ಆಧಾರಿತ EDA ಪರಿಕರಗಳೊಂದಿಗೆ ಅತ್ಯುತ್ತಮ ಸ್ಥಿರತೆಯನ್ನು ನೀಡುತ್ತದೆ.
- **ಕಾರ್ಯಕ್ಷಮತೆ**: ಇದು ಭಾರೀ ತಾಂತ್ರಿಕ ಸಂಶೋಧನೆಗಾಗಿ ದಕ್ಷ ಪಿಡಿಎಫ್ ನಿರ್ವಹಣೆ ಮತ್ತು ಮೆಮೊರಿ ನಿರ್ವಹಣೆಯನ್ನು ಒದಗಿಸುತ್ತದೆ.
- **ಕೆಲಸದ ಹರಿವು**: ಇದು ಈ VLSI ವರ್ಕ್‌ಸ್ಟೇಷನ್‌ನ ನಿರ್ದಿಷ್ಟ ಎಂಜಿನಿಯರಿಂಗ್ ಅಗತ್ಯಗಳೊಂದಿಗೆ ಹೊಂದಿಕೆಯಾಗುತ್ತದೆ.

### ನಿಮ್ಮ ಡೀಫಾಲ್ಟ್ ಬ್ರೌಸರ್ ಅನ್ನು ಹೇಗೆ ಬದಲಾಯಿಸುವುದು

ನೀವು ಬೇರೆ ಬ್ರೌಸರ್ ಅನ್ನು ಬಯಸಿದರೆ, MayankOS ಬದಲಾಯಿಸುವುದನ್ನು ಸುಲಭಗೊಳಿಸುತ್ತದೆ:
1. **ವೇರಿಯಬಲ್ ಬದಲಾಯಿಸಿ**: ನಿಮ್ಮ ಹೋಸ್ಟ್‌ನ `variables.nix` ಫೈಲ್ ಅನ್ನು ತೆರೆಯಿರಿ (ಉದಾಹರಣೆಗೆ, `hosts/msi-modern14c7m/variables.nix`) ಮತ್ತು `browser` ಲೈನ್ ಅನ್ನು ನಿಮ್ಮ ಆಯ್ಕೆಗೆ ಬದಲಾಯಿಸಿ (ಉದಾಹರಣೆಗೆ, `browser = "firefox";`).
2. **ಸ್ಥಾಪನೆಗಾಗಿ ಪರಿಶೀಲಿಸಿ**: ನಿಮ್ಮ ಆದ್ಯತೆಯ ಬ್ರೌಸರ್ `modules/core/packages.nix` ನಲ್ಲಿ ಪಟ್ಟಿಮಾಡಲಾಗಿದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ. ಅದು ಅಲ್ಲಿ ಇಲ್ಲದಿದ್ದರೆ, ಅದರ ಪ್ಯಾಕೇಜ್ ಹೆಸರನ್ನು (ಉದಾಹರಣೆಗೆ, `librewolf`) ಪಟ್ಟಿಗೆ ಸೇರಿಸಿ.
3. **ರೀಬಿಲ್ಡ್ (Rebuild)**: ಬದಲಾವಣೆಯನ್ನು ಅನ್ವಯಿಸಲು `mcli rebuild` ಅಥವಾ ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟ ರೀಬಿಲ್ಡ್ ಆಜ್ಞೆಯನ್ನು ಚಲಾಯಿಸಿ (ಉದಾಹರಣೆಗೆ, `sudo nixos-rebuild switch --flake .#amd`).

ನಾವು ಆಯ್ಕೆ ಮತ್ತು ಸ್ವಾತಂತ್ರ್ಯವನ್ನು ನಂಬುತ್ತೇವೆ. MayankOS ಅನ್ನು ನಿಮ್ಮ ವೈಯಕ್ತಿಕ ವರ್ಕ್‌ಸ್ಟೇಷನ್ ಆಗಲು ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ—ಅದನ್ನು ನಿಮ್ಮದಾಗಿಸಿಕೊಳ್ಳಲು ಮುಕ್ತವಾಗಿರಿ!

## 🛠️ ಕಸ್ಟಮ್ ಹಾರ್ಡ್‌ವೇರ್ ಮತ್ತು ಹೋಸ್ಟ್ ಸೆಟಪ್ ಮಾರ್ಗದರ್ಶಿ

1. **ಹೊಸ ಹೋಸ್ಟ್ ರಚಿಸುವುದು**:
   - `hosts/default` ಫೋಲ್ಡರ್ ಅನ್ನು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರ್ ಹೆಸರಿನ ಹೊಸ ಫೋಲ್ಡರ್‌ಗೆ ಕಾಪಿ ಮಾಡಿ (ಉದಾಹರಣೆಗೆ, `cp -r hosts/default hosts/my-laptop`).
2. **ನಿಮ್ಮ ಹಾರ್ಡ್‌ವೇರ್ ಕಾನ್ಫಿಗರೇಶನ್ ಅನ್ನು ಜನರೇಟ್ ಮಾಡುವುದು**:
   - ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟ ಹಾರ್ಡ್‌ವೇರ್ (ಡ್ರೈವ್‌ಗಳು, ಸಿಪಿಯು, ಇತ್ಯಾದಿ) ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚಲು `nixos-generate-config --show-hardware-config > hosts/your-hostname/hardware.nix` ರನ್ ಮಾಡಿ.
3. **ನಿಮ್ಮ ಪ್ರೊಫೈಲ್ ಆಯ್ಕೆ ಮಾಡುವುದು**:
   - `flake.nix` ಅನ್ನು ತೆರೆಯಿರಿ ಮತ್ತು ನಿಮ್ಮ ಹಾರ್ಡ್‌ವೇರ್‌ಗೆ ಹೊಂದಿಕೆಯಾಗುವಂತೆ `profile` ವೇರಿಯೇಬಲ್ ಅನ್ನು ಸೆಟ್ ಮಾಡಿ (ಆಯ್ಕೆಗಳು: `amd`, `intel`, `nvidia`, `nvidia-laptop`, `amd-nvidia-hybrid`, ಅಥವಾ `vm`).
4. **ವೇರಿಯೇಬಲ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು**:
   - ನಿಮ್ಮ ಪರದೆಯ ರೆಸಲ್ಯೂಶನ್, ಆದ್ಯತೆಯ ಶೆಲ್ (`barChoice`) ಮತ್ತು ಇತರ ವೈಯಕ್ತಿಕ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಸೆಟ್ ಮಾಡಲು `hosts/your-hostname/variables.nix` ಅನ್ನು ಎಡಿಟ್ ಮಾಡಿ.
5. **ಇತರ ಲ್ಯಾಪ್‌ಟಾಪ್‌ಗಳಿಗೆ ಬೆಂಬಲ**:
   - ನೀವು MSI ನಂತಹ ವಿಶೇಷ ಲ್ಯಾಪ್‌ಟಾಪ್ ಹೊಂದಿದ್ದರೆ, `msi-ec` ನಂತಹ ಕರ್ನಲ್ ಮಾಡ್ಯೂಲ್‌ಗಳನ್ನು ಹೇಗೆ ಸೇರಿಸುವುದು ಎಂಬುದರ ಉದಾಹರಣೆಗಳಿಗಾಗಿ ನೀವು `hosts/msi-modern14c7m/default.nix` ಅನ್ನು ನೋಡಬಹುದು.
6. **ಅಂತಿಮ ಮರುನಿರ್ಮಾಣ**:
   - ಎಲ್ಲವನ್ನೂ ಅನ್ವยಿಸಲು `sudo nixos-rebuild switch --flake .#your-profile` ರನ್ ಮಾಡಿ.

#### 🎹 ಪೈಪ್‌ವೈರ್ ಮತ್ತು ಅಧಿಸೂಚನೆ ಮೆನು ನಿಯಂತ್ರಣಗಳು

- ನಾವು ಲಿನಕ್ಸ್‌ಗಾಗಿ ಇತ್ತೀಚಿನ ಮತ್ತು ಅತ್ಯುತ್ತಮ ಆಡಿಯೊ ಪರಿಹಾರವನ್ನು ಬಳಸುತ್ತಿದ್ದೇವೆ. ಟಾಪ್ ಬಾರ್‌ನಲ್ಲಿ ಲಭ್ಯವಿರುವ ಅಧಿಸೂಚನೆ ಕೇಂದ್ರದಲ್ಲಿ ನೀವು ಮೀಡಿಯಾ ಮತ್ತು ವಾಲ್ಯೂಮ್ ನಿಯಂತ್ರಣಗಳನ್ನು ಹೊಂದಿರುತ್ತೀರಿ.

#### 🏇 ಆಪ್ಟಿಮೈಸ್ಡ್ ವರ್ಕ್‌ಫ್ಲೋ ಮತ್ತು ಸುಧಾರಿತ ವಿಂಡೋ ಮ್ಯಾನೇಜ್‌ಮೆಂಟ್

- **Hyprland ಬೆಂಬಲ**: ಹೆಚ್ಚಿನ ಸೊಬಗು ಮತ್ತು ದಕ್ಷತೆಗಾಗಿ ಡೀಫಾಲ್ಟ್ ಟೈಲಿಂಗ್ ವಿಂಡೋ ಮ್ಯಾನೇಜರ್.
- **Niri ಬೆಂಬಲ**: ಈಗ Niri ಗಾಗಿ ಪೂರ್ಣ ಬೆಂಬಲವನ್ನು ಒಳಗೊಂಡಿದೆ, ಇದು ಸ್ಕ್ರೋಲ್ ಮಾಡಬಹುದಾದ-ಟೈಲಿಂಗ್ ವೇಲ್ಯಾಂಡ್ ಸಂಯೋಜಕವಾಗಿದೆ. ನಿಮ್ಮ `variables.nix` ನಲ್ಲಿ `niriEnable` ಮೂಲಕ ಇದನ್ನು ಟಾಗಲ್ ಮಾಡಿ.
- **KDE Plasma (ಐಚ್ಛಿಕ)**: KDE Plasma 6 ಗಾಗಿ ಬೆಂಬಲ ಲಭ್ಯವಿದೆ ಆದರೆ ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ.
- ಇಲ್ಲಿ ಯಾವುದೇ ಬೃಹತ್ NeoVIM ಯೋಜನೆ ಇಲ್ಲ, ಅದ್ಭುತವಾದ NeoVIM ಸೆಟಪ್‌ಗಾಗಿ `nixvim` ಅನ್ನು ಬಳಸಲಾಗುತ್ತಿದೆ. ಈಗಾಗಲೇ ಭಾಷಾ ಬೆಂಬಲವನ್ನು ಸೇರಿಸಲಾಗಿದೆ.

#### 🖥️ ಮಲ್ಟಿ ಹೋಸ್ಟ್ ಮತ್ತು ಬಳಕೆದಾರ ಕಾನ್ಫಿಗರೇಶನ್

- ನೀವು ವಿಭಿನ್ನ ಹೋಸ್ಟ್ ಯಂತ್ರಗಳು ಮತ್ತು ಬಳಕೆದಾರರಿಗಾಗಿ ಪ್ರತ್ಯೇಕ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಬಹುದು.
- `modules/core/user.nix` ಫೈಲ್‌ನಲ್ಲಿ ನಿಮ್ಮ ಬಳಕೆದಾರರಿಗಾಗಿ ಹೆಚ್ಚುವರಿ ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸುಲಭವಾಗಿ ನಿರ್ದಿಷ್ಟಪಡಿಸಿ.
- ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸುಲಭವಾದ ಫೈಲ್ ರಚನೆ ಮತ್ತು ಸರಳ, ಆದರೆ ಸಮಗ್ರವಾದ ಕಾನ್ಫಿಗರೇಶನ್.

#### 👼 ಬೆಂಬಲದ ಮೇಲೆ ಕೇಂದ್ರೀಕರಿಸಿದ ಅದ್ಭುತ ಸಮುದಾಯ

- ಯಾವುದೇ ಸಹಾಯಕ್ಕಾಗಿ ಡಿಸ್ಕಾರ್ಡ್‌ನಲ್ಲಿ ಸಂಪರ್ಕಿಸಲು ಮುಕ್ತವಾಗಿರಿ.

#### 📦 ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸ್ಥಾಪಿಸುವುದು ಹೇಗೆ?

- ಪ್ಯಾಕೇಜ್‌ನ ಹೆಸರೇನು ಅಥವಾ ನೀವು ಎದುರಿಸಬಹುದಾದ ಕಾನ್ಫಿಗರೇಶನ್ ಅಡೆತಡೆಗಳನ್ನು ನಿವಾರಿಸುವ ಆಯ್ಕೆಗಳು ಲಭ್ಯವಿದೆಯೇ ಎಂದು ನೋಡಲು ನೀವು [Nix Packages](https://search.nixos.org/packages?) ಮತ್ತು [Options](https://search.nixos.org/options?) ಪುಟಗಳನ್ನು ಹುಡುಕಬಹುದು.
- ಪ್ಯಾಕೇಜ್ ಸೇರಿಸಲು `modules/core/packages.nix` ಮತ್ತು `modules/core/user.nix` ನಲ್ಲಿ ವಿಭಾಗಗಳಿವೆ. ಒಂದು ಸಿಸ್ಟಮ್ ವ್ಯಾಪ್ತಿಯಲ್ಲಿ ಲಭ್ಯವಿರುವ ಪ್ರೋಗ್ರಾಂಗಳಿಗಾಗಿ ಮತ್ತು ಇನ್ನೊಂದು ನಿಮ್ಮ ಬಳಕೆದಾರ ಪರಿಸರಕ್ಕಾಗಿ ಮಾತ್ರ.

#### 🐧 ಡೆಸ್ಕ್‌ಟಾಪ್ ಪರಿಸರವನ್ನು ಬದಲಾಯಿಸುವುದು

MayankOS ಬಹು ಪರಿಸರಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ:
- **Hyprland**: ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ ಸಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ.
- **Niri**: ನಿಮ್ಮ ಹೋಸ್ಟ್‌ನ `variables.nix` ನಲ್ಲಿ `niriEnable = true;` ಹೊಂದಿಸುವ ಮೂಲಕ ಇದನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ.
- **KDE Plasma**: KDE ಪ್ಲಾಸ್ಮಾವನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಲು, `modules/core/xserver.nix` ಗೆ ಹೋಗಿ ಮತ್ತು `services.desktopManager.plasma6.enable = true;` ಲೈನ್ ಮತ್ತು ಸಂಬಂಧಿತ `environment.systemPackages` ಬ್ಲಾಕ್ ಅನ್ನು ಅನ್ ಕಾಮೆಂಟ್ ಮಾಡಿ.

#### 🙋 ಸಮಸ್ಯೆಗಳು / ಪ್ರಶ್ನೆಗಳಿವೆಯೇ?

- ದಯವಿಟ್ಟು ರೆಪೊದಲ್ಲಿ ಸಮಸ್ಯೆಯನ್ನು ಎತ್ತಲು ಮುಕ್ತವಾಗಿರಿ, ದಯವಿಟ್ಟು ಫೀಚರ್ ವಿನಂತಿಯನ್ನು [feature request] ನಿಂದ ಪ್ರಾರಂಭವಾಗುವ ಶೀರ್ಷಿಕೆಯೊಂದಿಗೆ ಲೇಬಲ್ ಮಾಡಿ, ಧನ್ಯವಾದಗಳು!
- ವೇಗವಾದ ಪ್ರತಿಕ್ರಿಯೆಗಾಗಿ [Discord](https://discord.gg/XhZmNTnhtp) ನಲ್ಲಿಯೂ ನಮ್ಮನ್ನು ಸಂಪರ್ಕಿಸಿ.

<details>
<summary><strong> 🦽 ಹಸ್ತಚಾಲಿತ ಅನುಸ್ಥಾಪನಾ ಪ್ರಕ್ರಿಯೆ:  </strong></summary>

1. Git ಮತ್ತು Vim ಇನ್‌ಸ್ಟಾಲ್ ಆಗಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಲು ಈ ಕಮಾಂಡ್ ರನ್ ಮಾಡಿ:

```bash
nix-shell -p git vim
```

2. ಈ ರೆಪೊವನ್ನು ಕ್ಲೋನ್ ಮಾಡಿ ಮತ್ತು ಪ್ರವೇಶಿಸಿ:

```bash
cd && git clone https://github.com/techanand8/mayankos.git ~/mayankos
cd ~/mayankos
```

3. ನಿಮ್ಮ ಯಂತ್ರ(ಗಳ) ಗಾಗಿ ಹೋಸ್ಟ್ ಫೋಲ್ಡರ್ ಅನ್ನು ಈ ರೀತಿ ರಚಿಸಿ:

```bash
cp -r hosts/default hosts/<your-desired-hostname>
```

4. ನಿಮ್ಮ ಹಾರ್ಡ್‌ವೇರ್ ಕಾನ್ಫಿಗರೇಶನ್ ಅನ್ನು ರಚಿಸಿ:

```bash
nixos-generate-config --show-hardware-config > hosts/<your-desired-hostname>/hardware.nix
```

5. ನಿಮ್ಮ ಹೋಸ್ಟ್‌ನೇಮ್ ಮತ್ತು ಪ್ರೊಫೈಲ್‌ಗೆ ಹೊಂದಿಕೆಯಾಗುವಂತೆ `hosts/<your-desired-hostname>/variables.nix` ಮತ್ತು `flake.nix` ಅನ್ನು ಎಡಿಟ್ ಮಾಡಿ.

6. ಫ್ಲೇಕ್ ಅನ್ನು ಸ್ಥಾಪಿಸಿ (`profile` ಅನ್ನು `intel`, `nvidia`, `nvidia-laptop`, `amd`, `amd-nvidia-hybrid`, ಅಥವಾ `vm` ನೊಂದಿಗೆ ಬದಲಾಯಿಸಿ):

```bash
sudo nixos-rebuild switch --flake .#profile
```
