# 📚 Hojas de Trucos y Guías de MayankOS

Bienvenido a la biblioteca central de documentación de MayankOS. Estas guías están diseñadas para ayudarte a dominar tu nuevo entorno, desde conceptos básicos de NixOS hasta ingeniería avanzada de VLSI.

## 🚀 Guías Principales
- **[Guía para Principiantes de Nix](./nix-beginner-guide.es.md)**: Comienza aquí si eres nuevo en Nix/NixOS. Aprende a añadir paquetes y cambiar ajustes básicos.
- **[Guía del Proyecto](./project-guide.md)** (Inglés): Profundiza en la arquitectura de este repositorio.
- **[Personalización de Hyprland](./hyprland-customization-guide.es.md)**: Aprende a ajustar tu entorno de escritorio.
- **[Ingeniería de Hardware y VLSI](./vlsi-guide.es.md)**: Una guía de las herramientas EDA de grado industrial incluidas en MayankOS.
- **[Añadir Escritorio GNOME](./add-gnome.md)** (Inglés): Cómo habilitar el escritorio GNOME junto a Hyprland/Niri.

## 🛠️ Terminal y Herramientas
- **[Ghostty](./ghostty/ghostty.cheatsheet.es.md)**: Emulador de terminal moderno con aceleración por hardware.
- **[Tmux](./tmux/tmux.cheatsheet.es.md)**: Multiplexor de terminal para gestionar múltiples sesiones.
- **[Yazi](./yazi/yazi.keymap.cheatsheet.es.md)**: Gestor de archivos de terminal extremadamente rápido.
- **[Alacritty](./alacritty/alacritty.cheatsheet.es.md)** / **[Kitty](./kitty/kitty.cheatsheet.es.md)** / **[WezTerm](./wezterm/wezterm.cheatsheet.es.md)**: Emuladores de terminal alternativos.
